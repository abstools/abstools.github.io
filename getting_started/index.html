Getting Started

There are various ways of running ABS models.

Use On-Line Tools

The simplest way to use the ABS tools is on-line, in the collaboratory. This means you only need a modern browser to start experimenting with ABS. The tools work best with Firefox and Chrome.

An introduction and link to the collaboratory can be found at http://abs-models.org/laboratory/.

  • Installing Command-Line Tools

    • Many of the tools can be run from the command line. This page describes how to run various tools on a local machine.

  • Running a Browser-Based IDE

    • The collaboratory is a browser-based development environment for ABS. It integrates an editor, the syntax checker and simulator, and the SACO resource analysis tool.

  • Installing Editor Support

    • We provide support for Emacs, Atom and Visual Studio Code.
